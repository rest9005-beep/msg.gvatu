# ConnectFlow

Современный рабочий мессенджер с расширенными настройками приватности.

## Функции

- ✅ Реальная авторизация и регистрация
- ✅ Загрузка аватара (поддержка изображений до 5MB)
- ✅ Реальный поиск по юзернейму, имени и биографии
- ✅ Полная система приватности:
  - Видимость профиля (публичный/только друзья/приватный)
  - Кто может писать сообщения
  - Отображение статуса онлайн
  - Получение заявок в друзья
- ✅ Система друзей с заявками
- ✅ Чаты (базовая реализация)
- ✅ Современный дизайн с анимациями
- ✅ Полностью адаптивный интерфейс

## Установка

1. Создайте папку для проекта
2. Скопируйте все файлы в папку
3. Откройте `index.html` в браузере

## Использование

### Демо пользователи:

- maria.s / 123456 (только друзья могут писать)

### Поиск:
- Ищет по юзернейму, имени и биографии
- Учитывает настройки приватности
- Сортирует по релевантности

### Приватность:
- **Публичный**: видят все пользователи
- **Только друзья**: видят только друзья
- **Приватный**: никто не видит

## Технологии

- LocalStorage для хранения данных
- Font Awesome для иконок
- Google Fonts (Inter, JetBrains Mono)
- CSS Grid, Flexbox
- CSS Variables для темизации
